⚡ How to Play
1
Teams take turns picking any category and a point value from the board
2
Host sets the timer, presses Start — the question is revealed and the clock begins
3
Team must answer before time runs out. Correct answer earns the points
4
Host clicks Reveal Answer to confirm, then awards points via the scoreboard buttons
5
Each tile is used only once. Team with the most points at the end wins!
